# SplitAndSaveByUniqueValues---VBA-MACRO
This macro allows the user to: ' 1. Upload/select an Excel file. ' 2. Choose a column for filtering. ' 3. Identify all unique values in the selected column. ' 4. Filter data for each unique value. ' 5. Select a column range (start to end) for export. ' 6. Save each filtered dataset as a separate Excel file
